= Extrude to Pyramid or Cone Tool
:page-en: tools/Extrude_to_Pyramid_or_Cone
ifdef::env-github[:imagesdir: /en/modules/ROOT/assets/images]

Activate the tool, then:

* Select and drag a polygon or a circle to create a pyramid or cone.

or

* Select a polygon or a circle, then enter the value of the _height_ in the appearing input box to create a pyramid or cone.
The height can be positive or negative. In Classic v.5, the default setting is to have the positive direction of the altitude towards the user, indicated by the _Positive values follow the view_ checkbox being checked by default.

[NOTE]
====
A right pyramid or cone is created with the apex positioned above the centroid of the base.
====
