RegresjaKrzywejNiejawnej( <Lista Punktów>, <Kolejność> )::
  Próbuje znaleźć najlepiej dopasowaną krzywą zadaną niejawnie, o rządzie n ≥ 2, przechodzącą przez podane punkty. Potrzebujesz co najmniej stem:[\frac{n(n+3)}2]
  punktów.
