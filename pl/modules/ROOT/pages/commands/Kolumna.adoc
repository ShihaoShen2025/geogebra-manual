Kolumna( <Komórka Arkusza> )::
  Zwraca numer kolumny danej komórki (licząc od 1).

[EXAMPLE]
====

`++q = Kolumna(B3)++` daje w wyniku _q = 2_, ponieważ kolumna _B_ jest drugą kolumną
xref:/Widok_Arkusza.adoc[arkusza].

====
