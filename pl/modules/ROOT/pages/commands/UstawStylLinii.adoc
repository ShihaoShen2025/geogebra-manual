UstawStylLinii( <Obiekt>, <Liczba> )::
  Zmienia styl linii podanego obiektu zgodnie z poniższą tabelą (liczby spoza zakresu [0,4] są traktowane jako 0).

[cols=",",options="header",]
|===
|Liczba |Styl
|0 |Ciągła
|1 |Przerywana długa
|2 |Przerywana krótka
|3 |Kropkowana
|4 |Kreska-kropka
|===
