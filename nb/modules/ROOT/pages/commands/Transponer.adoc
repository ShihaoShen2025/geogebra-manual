Transponer[ <Matrise> ]::
  Transponerer matrisen.

[EXAMPLE]
====

`++Transponer[{{1, 2, 3}, {4, 5, 6}, {7, 8, 9}}]++` gir matrisen stem:[\begin{pmatrix}1&4&7\\ 2&5&8\\
3&6&9\end{pmatrix}].

====
