NastavitUrovenDetailu( <Plocha>, <Úroveň detailu> )::
  Nastaví, zda je plocha vykreslena rychle, méně detailně (úroveň detailu = 0), nebo pomalu, ale detailněji, tj. přesněji (úroveň detailu = 1).
