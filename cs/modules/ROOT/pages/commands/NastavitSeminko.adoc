NastavitSeminko( <Celé číslo> )::
  Nastaví počáteční hodnotu pro generátor náhodných čísel. Následná náhodná čísla jsou touto počáteční hodnotou (semínkem, angl. seed) určena.

[EXAMPLE]
====

`++NastavitSeminko(33)++`

====

====
