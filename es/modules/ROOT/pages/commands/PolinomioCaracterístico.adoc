== Sintaxis CAS

PolinomioCaracterístico( <Matriz> )::
  Devuelve el https://es.wikipedia.org/wiki/Polinomio_caracter%C3%ADstico[polinomio característico] de la matriz dada.

[EXAMPLE]
====

`++PolinomioCaracterístico({{1,2},{3,4}})++` devuelve stem:[x^2-5x-2].

====
