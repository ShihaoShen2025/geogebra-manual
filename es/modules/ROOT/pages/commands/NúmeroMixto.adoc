== CAS Syntax

NúmeroMixto( <Número> )::
  Expresa el número dado como número mixto.

[EXAMPLE]
====

*Ejemplos:*

* `++NúmeroMixto(3.5)++` da por resultado stem:[3 + \frac{1}\{2}].
* `++NúmeroMixto(12 / 3)++` da por resultado _4_.
* `++NúmeroMixto(12 / 14)++` da por resultado stem:[\frac{6}\{7}].

====

====
