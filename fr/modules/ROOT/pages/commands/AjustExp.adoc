AjustExp(<Liste Points>)::
  Recherche une courbe de régression exponentielle de la forme stem:[ a e ^{ b x } ].

[EXAMPLE]
====

`++AjustExp({(0, 1), (2, 4)})++` crée stem:[1e ^{0.69 x}].

====
